# بيانات الدخول 🔐

## حسابات موظفي المحل

استخدم هذه الحسابات لتسجيل الدخول إلى لوحة تحكم الموظفين:

### حساب المدير (Admin)
```
الإيميل: admin@kingshop.com
كلمة المرور: admin123
الصلاحيات: إدارة كاملة
```

### حساب الحلاق (Barber)
```
الإيميل: barber@kingshop.com
كلمة المرور: barber123
الصلاحيات: عرض ومعالجة الحجوزات
```

### حساب الموظف (Staff)
```
الإيميل: staff@kingshop.com
كلمة المرور: staff123
الصلاحيات: عرض الحجوزات
```

---

## خطوات تسجيل الدخول

1. اذهب إلى صفحة تسجيل الدخول
2. أدخل الإيميل
3. أدخل كلمة المرور
4. اضغط "دخول"
5. ستصل إلى لوحة تحكم الموظفين 📊

---

## ملاحظات مهمة ⚠️

- كلمات المرور مشفرة في قاعدة البيانات
- يمكن إضافة حسابات جديدة من صفحة إدارة الموظفين
- يمكن تغيير الصلاحيات (staff / manager / admin) من صفحة الإدارة
- يمكن تفعيل/تعطيل الحسابات من لوحة التحكم

---

## للدخول أولاً 🚀

1. تأكد من أن قاعدة البيانات تم تشغيل schema.sql
2. اذهب إلى: https://kingshop.vercel.app/login
3. استخدم أي من الحسابات أعلاه
4. ستظهر لك لوحة تحكم الحجوزات
